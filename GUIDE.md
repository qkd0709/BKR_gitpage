# 📝 나만의 대시보드 페이지 수정 가이드

이 파일은 `index.html` 파일을 직접 수정하여 페이지의 내용을 바꾸는 방법을 설명합니다.

---

## 1. 기본 정보 수정 (이름, 소속 등)
`index.html` 파일에서 `id="info"` 섹션을 찾으세요 (약 50번 라인).
- `<strong>이름:</strong>` 뒤의 텍스트를 본인 이름으로 바꿉니다.
- `학번/소속`, `이메일` 등도 동일하게 수정합니다.

## 2. 수강 과목 수정 및 추가
`id="courses"` 섹션을 찾으세요 (약 75번 라인).
- **수정**: `<h3>` 안의 과목명과 `<p>` 안의 설명을 바꿉니다.
- **추가**: `<div class="card"> ... </div>` 블록을 복사해서 바로 아래에 붙여넣으면 새 과목이 생깁니다.

## 3. 배운 언어 숙련도 수정
`id="languages"` 섹션을 찾으세요 (약 95번 라인).
- **막대 바 길이**: `<div class="progress" style="width: 85%;">` 부분에서 숫자를 조절하세요 (0~100).
- **텍스트**: `<span>` 안의 언어 이름과 `level` 안의 숙련도 텍스트를 바꿉니다.

## 4. 과제 내역 관리 (가장 중요!)
`id="assignments"` 섹션을 찾으세요 (약 130번 라인).
표의 각 행은 `<tr> ... </tr>`로 구성됩니다.

### 내용 수정
`<td>` 태그 안의 텍스트를 본인의 과제 내용으로 바꿉니다.

### 상태 배지(Badge) 색상 변경
`<span class="badge ...">` 부분에서 클래스명을 바꾸면 색상이 변경됩니다.
- `badge success`: 초록색 (완료)
- `badge warning`: 노란색 (진행 중)
- `badge info`: 파란색 (대기)
- `badge error`: 빨간색 (미착수)

### 과제 추가
`<tr> ... </tr>` 블록 하나를 통째로 복사해서 `<tbody>` 태그 안에 추가하세요.

## 5. 이미지 교체
- **아바타**: 본인의 사진을 `avatar.png`라는 이름으로 저장하여 폴더에 덮어쓰기 하거나, `index.html`에서 `src="avatar.png"` 부분을 본인의 이미지 파일명으로 바꿉니다.

---

### 💡 주의사항
1. **저장**: 수정 후에는 반드시 `Ctrl + S`를 눌러 저장하세요.
2. **새로고침**: 브라우저에서 `F5`를 눌러 변경 사항을 확인하세요.
3. **태그 주의**: `<` 나 `>` 같은 기호를 실수로 지우면 디자인이 깨질 수 있으니 주의하세요!
